# X-Nonce
The X-Nonce header is a custom HTTP header often used in web applications as part of security measures, particularly in the context of preventing CSRF (Cross-Site Request Forgery) attacks.

usecase: https://nextjs.org/docs/app/building-your-application/configuring/content-security-policy#nonces
