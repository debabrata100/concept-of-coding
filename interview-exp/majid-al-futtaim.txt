6. Difference between useMemo and useCallback ?
useMemo memoizes the result of a computation, while useCallback memoizes the function itself. Both are used for performance optimization, but should only be applied when there’s a measurable benefit, such as preventing unnecessary re-renders or avoiding expensive recalculations.
Don’t overuse — both add memory + comparison overhead, so only use when there’s a clear performance gain. In many cases, React’s built-in optimizations are sufficient without needing to memoize.

    let res = {};
  
    arr.forEach((el) => {
      if (!res[el[key]]) {
        console.log(res[el[key]]);
        res[el[key]] = {};
        for (let prop in el) {
          if (prop !== key) {
            res[el[key]][prop] = [el[prop]];
          }
        }
      } else {
