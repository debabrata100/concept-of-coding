1. What is TypeScript, and how does it differ from JavaScript?
TypeScript is a superset of JavaScript developed by Microsoft. It adds static typing and other features to the language, which helps in writing more robust, maintainable, and scalable applications.

🔑 Key Differences Between TypeScript and JavaScript
Feature	        JavaScript	                        TypeScript
Typing	        Dynamically typed	                Statically typed (optional and can be inferred)
Compilation	    Interpreted directly by browsers	Compiled to JavaScript before execution
Error Detection	Runtime errors	                    Compile-time type checking to catch errors early

8. Explain the concept of type inference in TypeScript.
If you don’t explicitly write a type, TypeScript will guess it based on the assigned value.
let isLoggedIn = true; // TypeScript infers: boolean
isLoggedIn = "yes";    // ❌ Error: 'string' is not assignable to 'boolean'

12. What is the purpose of "tsconfig.json" file in a TypeScript project?
The tsconfig.json file is a critical configuration file in any TypeScript project. It tells the TypeScript compiler (tsc) how to compile your code.
