    Example 2: Extracting Dates
    You can use regex to extract dates from a text.

    const text = "The meeting is scheduled for 2023-11-15. Don't forget!";
    const datePattern = /\d{4}-\d{2}-\d{2}/;
    const dates = text.match(datePattern);

    Example 3: Replacing Words
    Regex can be used for replacing words in a text with other words.
    const text = "I like cats, but I prefer dogs.";
    const replacePattern = /cats/g;
    const newText = text.replace(replacePattern, "dogs");
